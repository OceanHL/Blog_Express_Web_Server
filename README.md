# Express搭建的博客Web Server

## 技术特点

- `TypeScript`、`NodeJS`、`Express`、`MySQL` 数据库、`redis` 数据库、`pm2`。

## 功能

- 日志功能
- 登录校验【`redis`、`session`】
- 博客的增删改查
- `pm2` 管理项目

## 启动项目

### 第1步：安装依赖

```js
yarn install
# or
npm i
```

### 第2步：输入 `yarn dev` 或 `yarn pro` 命令

```js
# 开发环境
yarn dev

# 生产环境
yarn pro
```
